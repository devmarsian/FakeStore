The Fake Store is a mobile shopping interface that allows users to browse a selection of products from a fictional online store. Built using Kotlin Multiplatform. 

Technologies used: 
- Kotlin Multiplatform 
- Jetpack Compose
- MVIKotlin
- Decompose
- Ktor
